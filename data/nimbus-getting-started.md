# Nimbus — Getting Started

## Your first note
Press `Cmd/Ctrl + N` to create a note. Nimbus uses Markdown, so `#` makes a
heading, `-` a list, and `[[double brackets]]` link to another note. Links are
created automatically if the target note doesn't exist yet.

## Organising with tags and notebooks
Add `#tags` anywhere in a note to group ideas across notebooks; a notebook is a
folder for a project or area. A note can live in one notebook but carry many
tags, so you don't have to choose a single place for it.

## Search
`Cmd/Ctrl + K` opens search. It matches note titles, body text and tags, and
ranks recently-opened notes higher. On Pro and above, search also works fully
offline against the local copy.

## Syncing devices
Sign in on a second device and your workspace appears automatically. Sync is
local-first: edits apply instantly on-device and reconcile in the background,
so there's no "save" button and no spinner while you write.

## Importing from another app
Settings → Import supports Markdown folders, Evernote `.enex`, and Notion
exports. Links and attachments are remapped so your existing structure survives
the move.
